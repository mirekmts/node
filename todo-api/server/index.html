Witam w Todo API.

Całość jest napisana w NodeJS i zapisuje się w bazie danych Mongodb.


Pod routem /todos - są dostępne wszystkie zadania do wykonania.

Pod routem /todos/:id - mamy konkretne jedno zadania.


Możemy korzystać z GET, POST, DELETE, PATCH. Najprściej będzie skorzystać z POSTMAN'a, aby dodać/usunąć/zaaktaulizować dane.
